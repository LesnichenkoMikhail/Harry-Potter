# Harry-Potter

#  Harry Potter Text Explorer (Shiny NLP Project)

Интерактивное Shiny-приложение для анализа текстов книг серии **Harry Potter** с использованием методов NLP и Machine Learning.

Проект включает:
- анализ тональности текста
- TF-IDF анализ ключевых слов
- сеть персонажей (co-occurrence graph)
- классификацию текста по книге (ML модель)
- интерактивную визуализацию данных

##  Описание проекта

Цель проекта — исследовать текстовую структуру серии книг *Harry Potter* с помощью методов цифровых гуманитарных наук (Digital Humanities).

Мы анализируем:
- эмоциональную динамику текста
- ключевые слова каждой книги
- взаимодействия персонажей
- возможность автоматического определения книги по фрагменту текста

---

##  Архитектура проекта

Проект реализован на **R Shiny** и имеет модульную структуру.


##  Основные компоненты

---

###  1. Sentiment Analysis (`sentiment.R`)

Анализ эмоциональной окраски текста с использованием словаря Bing.

 Результат:
- график эмоциональной динамики по главам
- сравнение книг

---

###  2. TF-IDF Analysis (`tfidf.R`)

Определение ключевых слов каждой книги.

 Результат:
- топ-15 уникальных слов для каждой книги
- сравнение лексики между книгами

---

###  3. Character Network (`network.R`)

Построение графа совместного появления персонажей.

 Узлы:
- персонажи (Harry, Ron, Hermione и др.)

 Рёбра:
- совместное упоминание в одном текстовом фрагменте

 Возможности:
- фильтрация по книге
- регулировка силы связей
- интерактивная визуализация (visNetwork)

---

### 🤖 4. Classifier (`classifier.R` + `global.R`)

ML-модель для определения книги по текстовому фрагменту.

 Pipeline:
- tokenization
- stopwords removal
- TF-IDF
- multinomial logistic regression (glmnet)

 Результат:
- предсказание книги (Book 1–7)
- интерактивный ввод текста

- TF-IDF через `textrecipes`
- графы через `igraph` + `visNetwork`
- ML через `tidymodels + glmnet`
